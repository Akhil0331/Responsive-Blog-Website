# Responsive-Blog-Website
# Project Description
A Responsive Blog Website is a web application where users can read blog posts, explore categories, and interact with content in a clean, mobile-friendly interface. The design automatically adjusts to different screen sizes such as mobile, tablet, and desktop, ensuring a smooth user experience everywhere.
This project focuses on building a simple yet professional-looking blog layout using HTML, CSS, and JavaScript, with an emphasis on responsiveness using CSS Flexbox, Grid, and media queries. The website contains a homepage with featured posts, a blog listing page, and a single post page.

# Technologies Used
HTML5 – page structure
CSS3 – styling, layout, responsiveness
JavaScript – interactivity (menu toggle, search, animations)
Google Fonts / Icons – UI enhancement

# Objectives
To design a fully responsive blog website using modern front-end technologies.
To implement different layouts for desktop, tablet, and mobile screens.
To demonstrate understanding of CSS Flexbox, Grid, and media queries.
To provide a simple blogging interface with a clean and user-friendly design.

# System Workflow
User opens the homepage and sees featured posts.
Navigation menu allows switching to categories or blog pages.
User selects a blog post.
Single blog post page displays full article content.
Responsive layout ensures proper appearance on all devices.
